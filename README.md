# Body Challenge

Групповой веб-челлендж: каждый участник закрашивает силуэт тела за дневную активность и соревнуется в **рейтинге недели**.

## Что умеет приложение

- Регистрация и вход (email + пароль)
- Личный прогресс за сегодня (тело + очки)
- Быстрое добавление активности (отжимания, шаги, зарядка и т.д.)
- **Общий рейтинг недели** — кто набрал больше очков
- Работает в браузере на телефоне и компьютере

## Бесплатные сервисы

| Сервис | Зачем | Цена |
|--------|-------|------|
| [Supabase](https://supabase.com) | аккаунты + база данных | бесплатно |
| [GitHub](https://github.com) | хранение кода | бесплатно |
| [Vercel](https://vercel.com) или [Netlify](https://netlify.com) | публикация в интернете | бесплатно |

Для группы из 5–30 человек этого хватит.

## Шаг 1. Supabase (база и пользователи)

1. Зарегистрируйтесь на [supabase.com](https://supabase.com)
2. **New project** → выберите имя и пароль БД
3. Откройте **SQL Editor** → **New query**
4. Скопируйте содержимое файла `supabase/schema.sql` и нажмите **Run**
5. Откройте **Authentication → Providers → Email** и при желании отключите **Confirm email** (удобнее для друзей на старте)
6. Откройте **Project Settings → API** и скопируйте:
   - **Project URL**
   - **anon public key**

## Шаг 2. Настройка приложения

Откройте `js/config.js` и вставьте ваши ключи:

```js
export const SUPABASE_URL = 'https://xxxxx.supabase.co';
export const SUPABASE_ANON_KEY = 'eyJhbGciOi...';
```

## Шаг 3. Локальный просмотр

Можно открыть `index.html` напрямую, но для модулей ES лучше простой сервер:

```bash
# если установлен Python
python -m http.server 8080

# или npx (если установлен Node.js)
npx serve .
```

Откройте `http://localhost:8080`

## Шаг 4. Публикация в интернете

### Вариант A: Vercel

1. Создайте репозиторий на GitHub и загрузите код
2. Зайдите на [vercel.com](https://vercel.com) → **Add New Project**
3. Import Git Repository → выберите репозиторий
4. Deploy

Получите ссылку вида: `https://body-challenge.vercel.app`

### Вариант B: Netlify

1. Загрузите папку на [netlify.com](https://netlify.com) через **Deploy manually**
2. Или подключите GitHub репозиторий

### Вариант C: GitHub Pages

1. Settings → Pages → Source: main branch
2. Ссылка: `https://username.github.io/body-challenge`

## Как пользоваться (для участников)

1. Откройте общую ссылку
2. Зарегистрируйтесь (имя будет в рейтинге)
3. Добавляйте активность в течение дня
4. Смотрите, кто лидирует в **рейтинге недели**
5. В понедельник неделя начинается заново

## Формула очков

| Активность | Очки |
|------------|------|
| 1 отжимание | 2 |
| 1 приседание | 1.5 |
| 1000 шагов | 5 |
| 1 мин зарядки | 3 |
| 1 км бега | 10 |
| 1 сек планки | 0.5 |

100 очков = 100% тела за день.

## Безопасность

- Каждый видит только свой список активностей за день
- Рейтинг показывает только имя и сумму очков
- Row Level Security в Supabase не даёт редактировать чужие данные
